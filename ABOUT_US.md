# About FlitPDF

**Last Updated:** April 20, 2026

FlitPDF is an open-source mobile app focused on practical PDF and document workflows. It is built to help users scan, create, compress, view, and organize files directly on their devices.

## What FlitPDF Offers

- Document scanning
- Image to PDF conversion
- PDF viewing
- PDF compression
- Image compression
- Recent file access and local document utilities

## Open Source

FlitPDF welcomes community contributions. If you want to fix bugs, improve performance, expand tooling, or polish the user experience, pull requests and well-scoped issues are encouraged.
